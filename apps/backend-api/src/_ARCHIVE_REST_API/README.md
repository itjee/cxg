# RESTful API 아카이브

이 폴더는 기존 RESTful API 구조를 보관합니다.

## ⚠️ 중요

이 코드는 **더 이상 사용하지 않습니다**.

모든 API는 GraphQL v3.0 구조로 개발되고 있습니다:
- {시스템명}/{스키마명}/{엔티티명}
- 예: manager/idam/users, tenants/sys/users

## 📦 보관된 내용

- **routers/** - RESTful API 라우터
- **modules/** - 비즈니스 로직 모듈

## 🎯 새로운 구조

GraphQL v3.0 구조는 `src/graphql/` 폴더를 참조하세요.
