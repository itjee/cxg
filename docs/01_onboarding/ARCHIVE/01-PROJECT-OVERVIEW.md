# 프로젝트 개요

## 프로젝트 정보

### 기본 정보

- **프로젝트명**: CXG Platform (AI 기반 업무지원 플랫폼)
- **제품 유형**: SaaS (Software as a Service)
- **대상 고객**: 50인 미만 중소기업
- **핵심 가치**: ERP, CRM, SCM, WMS, 전자결재 등을 하나의 통합 플랫폼으로 제공

### 시스템 구성

- **관리자 시스템** (apps/web-mgmt): 테넌트 관리, 인프라 모니터링, 청구/과금
- **통합 백엔드 API** (apps/api): FastAPI 기반 REST API 서버
- **테넌트 시스템** (apps/web-tnnt): 업무 처리 시스템 (ERP, CRM, SCM 등)

## 비즈니스 도메인

### 주요 사용자

1. **관리자 (운영자)**: 시스템 전체 관리, 테넌트 관리, 모니터링
2. **경영진 (테넌트)**: 경영 대시보드, KPI 모니터링, 의사결정 지원
3. **실무진 (테넌트)**: 일상 업무 처리, 데이터 입력/조회
4. **외부 파트너**: API 연동, 데이터 교환

### 핵심 워크플로우

1. **고객 관리**: 리드 → 영업기회 → 견적 → 계약 → 고객관리
2. **주문 처리**: 견적 → 주문 → 재고확인 → 출고 → 배송 → 매출인식
3. **구매 관리**: 구매요청 → 승인 → 발주 → 입고 → 매입 → 지급
4. **재무 관리**: 거래발생 → 전표생성 → 장부기록 → 채권채무관리 → 결산

## 주요 모듈

### 관리자 시스템 모듈

- **TNNT**: 테넌트 관리 (생성, 구독, 온보딩)
- **IFRA**: 인프라 관리 (리소스, 사용량)
- **IDAM**: 사용자/권한 관리 (사용자, 역할, 권한, 세션)
- **BILL**: 청구 관리 (요금제, 청구서, 결제)
- **MNTR**: 모니터링 (헬스체크, 장애, 메트릭)
- **AUDT**: 보안/감사 (감사로그, 컴플라이언스, 정책)
- **STAT**: 분석/통계 (테넌트 통계, 사용량 분석)
- **SUPT**: 지원 (티켓, 피드백)
- **CNFG**: 설정 (시스템 설정, 기능 플래그)
- **BKUP**: 백업/복구 (백업 관리, 재해복구)
- **NOTI**: 알림 (알림 관리, 템플릿)
- **INTG**: 외부 연동 (API, 웹훅, Rate Limit)
- **AUTO**: 자동화 (워크플로우, 스케줄 작업)

### 테넌트 시스템 모듈

- **ADM**: 기준정보 (회사, 부서, 사원, 거래처, 제품, 창고)
- **PSM**: 구매/조달 (구매요청, 발주, 입고, 매입, 반품)
- **SRM**: 영업/매출 (영업기회, 견적, 주문, 출고, 매출)
- **IVM**: 재고/자재 (재고현황, 이동, 조정, 예약, 월령)
- **LWM**: 물류/창고 (입고, 출고, 배송, 3PL)
- **CSM**: 고객지원 (티켓, FAQ, 지식베이스)
- **ASM**: A/S 관리 (접수, 처리, 부품, 보증)
- **FIM**: 재무/회계 (채권채무, 자산, 원가, 손익)
- **BIM**: 경영분석 (KPI, 목표, 분석, 예측)
- **COM**: 공통/지원 (전자결재, 캘린더, 지식관리)
- **SYS**: 시스템 관리 (사용자, 권한, 메뉴, 감사로그)

## 개발 원칙

### 핵심 원칙

1. **타입 안전성 우선**: TypeScript strict 모드, Python 타입 힌트 필수
2. **에러 핸들링 필수**: 모든 API와 컴포넌트에 적절한 에러 처리
3. **성능 최적화**: 페이징, 캐싱, 지연 로딩 고려
4. **보안 강화**: 인증/권한 체크, 입력 검증
5. **테스트 커버리지**: 핵심 비즈니스 로직은 반드시 테스트

### 우선순위

1. 백엔드 API 안정성
2. 프론트엔드 사용자 경험
3. 데이터 일관성
4. 시스템 성능
5. 코드 품질

### 절대 금지 사항

- 하드코딩된 설정값 사용
- SQL 인젝션 가능성 있는 코드
- localStorage 사용 (아티팩트에서 미지원)
- 타입 체크 우회 (any 타입 남용)
- 에러 핸들링 생략

## 개발 로드맵

- 인프라 구축

- 기본 대시보드

- 재고/물류 관리

- 전자결재
- 모바일 앱

### Phase 3: Advanced Features (3개월)

- AI 어시스턴트
- 고급 분석/BI
- A/S 관리
- 외부 시스템 연동
- 고급 자동화

### Phase 4: Enterprise (계속)

- 다국어 지원
- 고급 커스터마이징
- 엔터프라이즈 기능
- 파트너 에코시스템
